Detailed Description

This pass finds functions which are adapters to other functions. They typically either directly call some other function or do some simple arithmetics and then make the call.

This pass MUST run after -instcombine to make instruction matching simple (possible).

Right now only one known patter is implemented. When other patterns are found implement them here as separate functions.
