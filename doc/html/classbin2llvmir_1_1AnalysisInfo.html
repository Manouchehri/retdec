Detailed Description

Contains some support for info that is contained by analysis.

Contains support for:

  • For extended right uses.
  • For not go through.
  • For last left uses.
  • For global variables for indirect calls.
  • For global variables for calls for functions defined out of module.
  • For more details about this info
    See also
    StoreLoadAnalysis
