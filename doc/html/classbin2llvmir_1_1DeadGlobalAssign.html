Detailed Description

Optimization that removes dead global assigns.

This optimization optimizes for example (g and v are global variables):

g = 1; -- remove, not use this value.
g = 2;
v = 2; -- remove, only if v is not used anywhere from this place (also not
used in called functions).
x = g;

Also removes all definitions of global variables that don't have use.

This optimization can run in two variants. Aggressive and not aggressive. Not aggressive is run by:

-dead-global-assign -not-aggressive

Aggressive variant is default. Aggressive variant does not count that there is some use in functions that are defined out of the module.

This optimization can be run with statistics about how many global declaration or how many dead global assigns were deleted. This is possible with:

--stats -dead-global-assign
