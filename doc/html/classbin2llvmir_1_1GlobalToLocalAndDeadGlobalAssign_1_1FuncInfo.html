◆ findPatterns()

void bin2llvmir::GlobalToLocalAndDeadGlobalAssign::FuncInfo::findPatterns ( )

Finds the patterns in function.

Pattern is something like that.

func() {
tmp = g; // Part of pattern.
g = 4;
printf("%d", g);
g = 5;
printf("%d", g);
g = tmp; // Part of pattern.
}

In this situation we can change global variable g to local and remove assign to tmp variable and from this tmp variable.
