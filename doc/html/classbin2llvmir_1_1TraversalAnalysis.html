Detailed Description

Base class for traversal analyses.

This class contains shared implementation for traversal analyses so you can't create instance of this class. You have to use specific traversal.

bool bin2llvmir::TraversalAnalysis::causeNextNewSCCIteration ( )

Returns true if next item to return will cause new iteration of SCC, otherwise false.

Use only if you are sure that you iterates through the items that are part of some SCC. To find out it use isNextInSCC().
