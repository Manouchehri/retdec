Detailed Description

Analysis of variable dependency of PHI Nodes.

This class supports two things:

  • Detects the cycles and return PHI nodes that have to be optimized to remove the cycles.
  • Makes an analysis that returns PHI nodes in order that sequential processing of PHI nodes is equivalent with parallel processing.

Creates edge that describes dependency of variables.

For example:

%A = [ %B, %.bb1].

We create edge

B -> A with label %.bb1(basic block)

and saves PHI node to node A.

Parameters
[in]srcNodeNameName of source node.
[in]dstNodeNameName of destination node.
[in]incBBBasic block that describes relation.
[in]phiNodePHI node that is used to create this edge.
