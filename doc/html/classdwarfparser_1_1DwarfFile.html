◆ makeStructTypesUnique()

void dwarfparser::DwarfFile::makeStructTypesUnique ( )
private

Same-named structures may exist. We need to rename them -> make them unique, otherwise, string based (LLVM) type representation can not be used. Several same named structures would be generated and it would not be possible to distinguish their uses from one another.
