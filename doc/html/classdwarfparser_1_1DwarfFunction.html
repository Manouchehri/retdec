Compute and return value of function's frame base.

Parameters
nPointer to string that will be filled by method. If location is address this is a name of address space. If location is register this is a name of register array.
aPointer to value that will be filled by method. If location is address this is an address in address space. If location is register this is a number in register array.
pcProgram counter value.
Returns
Type of location - address or register.
Note
