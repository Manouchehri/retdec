bool dwarfparser::DwarfLocationDesc::isOnStack ( Dwarf_Signed *  off,
bool *  deref,
Dwarf_Addr  pc = 0,
int *  regNum = nullptr 
)

If variable is on stack, get its offset from stack pointer;.

Parameters
offPointer to offset that will be filled by method. If variable is not on stack it will be set to zero.
derefPointer to boolean value that will be filled by method. If true then on address counted using returned offset is an address of variable data. If false then there are actual data on counted address. If variable is not on stack it will be set to false.
pcActual program counter.
regNumPointer to number of register where the offset is, filled by method. -1 if on stack. This is used in case of DW_OP_bregX.
Returns
True if variable on stack, false otherwise.
