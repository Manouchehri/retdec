◆ getValueByName()

llvm::Value* llvm_support::tests::LlvmIrTests::getValueByName ( const std::string &  n)
inlineprotected

Utility function to get value (i.e. global variable, function, instruction) with the specified n from module m. Values are searched in the following order:

  1. globals
  2. functions
  3. instructions The first match is returned, therefore it is best to make sure all values in module have unique names.

Typical usage:

  1. create module using parseInput()
  2. if you need some specific values to work with, get them using this function. This is preferable to:
  1. create module using addModuleToContext()
  2. manually create objects and add them to module because manual object creation is not pretty – it is very hard to see what the resulting module looks like.
