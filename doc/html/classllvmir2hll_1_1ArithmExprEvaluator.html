Detailed Description

A base class for all evaluators.

A concrete evaluator should

  • implement the virtual functions where you wan't to change behaviour of sub-evaluator. Default implementation of this virtual function do nothing.
  • define a static ShPtr<ArithmExprEvaluator> create() function
  • register itself at ArithmExprEvaluatorFactory by passing the static create function and the concrete evaluator's ID
