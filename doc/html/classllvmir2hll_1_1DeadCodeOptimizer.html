Detailed Description

Removes dead code. Examples are mentioned below.

WhileLoopStmt
while (false) {
statement;
}
statement2;
can be optimized to
statement2;
IfStmt
if (true) {
statement;
}
can be optimized to
statement;
if (false) {
statement;
} else {
statement2;
}
can be optimized to
statement2;
if (true) {
statement;
} else if (false) {
label: statement2;
} else if (anything) {
statement3;
} else if (false) {
statement4;
}
can be optimized to
if (true) {
statement;
} else if (false) {
label: statement2;
}
SwitchStmt
switch (2) {
case 2: statement; break;
}
can be optimized to
statement;
switch (2) {
case 2: statement; break;
case 4: label: statement; break;
case 8: statement; break;
}
can be optimized to
switch (2) {
case 2: statement; break;
case 4: label: statement; break;
}

Instances of this class have reference object semantics.

This is a concrete optimizer which should not be subclassed.
