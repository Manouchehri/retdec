Detailed Description

A general semantics for the GCC compiler.

This class provides a general semantics for the GCC compiler. It may return implementation-defined results. The data for it were obtained from a GNU/Linux system (Arch Linux 64b, kernel 3.8.6), running gcc 4.8.0 and glibc 2.17.

Instances of this class have reference object semantics.
