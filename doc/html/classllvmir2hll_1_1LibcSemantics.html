Detailed Description

Semantics for the standard C library.

This class provides the semantics for the standard C library.

Instances of this class have reference object semantics.

Gets the information whether the given function never returns.

Returns
Just(true) if the given function never returns, Just(false) if the function may return, and Nothing<bool>() otherwise.
Parameters

For example, semantics describing the standard C library may return Just(true) for "exit".
