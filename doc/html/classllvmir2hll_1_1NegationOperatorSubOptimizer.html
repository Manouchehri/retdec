Detailed Description

This optimizer optimizes negation operators outside of expressions.

Optimization is on the NegOpExpr operator (i.e. ! in C).

List of performed simplifications (by examples):

Operator !
if (!(legume == 0 || legume > 4)) {...}
can be optimized to
if (legume != 0 && legume <= 4)) {...}
or
if (!(legume == 0 && legume > 4)) {...}
can be optimized to
if (legume != 0 || legume <= 4)) {...}
or
a = !!a
can be optimized to
a = a

Instances of this class have reference object semantics.

This is a concrete sub-optimizer which should not be subclassed.
