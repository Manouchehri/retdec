Detailed Description

The original converter of LLVM IR into BIR.

Instances of this class have reference object semantics.

Generates an if statement for a conditional branch to bb1 and bb2, depending on cond, including a goto statement(s).

Parameters
[in]bb1First target of the branch instruction.
[in]bb2Second target of the branch instruction.
[in]sourceSource basic block from which we are branching to bb1 and bb2.
[in]condBranch condition.
[in]negateCondIf true, it negates cond.

If a goto to bb2 is also required, it is generated instead of generating bb2 directly.

Preconditions
  • mapHasKey(bbStmtMap, bb1)
  • branchInfo->isGotoNecessary(source, bb1)

Returns
true if some PHI copies were generated, false otherwise.
