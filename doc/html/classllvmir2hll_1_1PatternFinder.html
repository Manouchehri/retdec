Detailed Description

A base class for all pattern finders.

A concrete finder should

  • implement all the pure virtual functions
  • define a static ShPtr<PatternFinder> create() function
  • register itself at PatternFinderFactory by passing the static create function and the finder's ID

Note: Do NOT set the ID of your concrete finder to "all" as this ID is reserved.

Instances of this class have reference object semantics.
