Detailed Description

A base class for all descriptions of function semantics.

This class should be used as a base class for all semantics.

To implement a new semantics, you should:

  • create a new class that subclasses this class or its subclass (you can copy and edit an existing semantics)
  • override the needed virtual member functions
  • create a static create() function and register the semantics at SemanticsFactory (see the implementation of existing subclasses)

Instances of this class have reference object semantics.
