ShPtr< Variable > llvmir2hll::VariablesManager::getVarByName ( const std::string &  name)
private

Returns the variable named by name.

If resulting module contains function named by name, this function is returned as variable. If resulting module contains global variable named by name, this global variable is returned. Otherwise, variable named by name is returned.

If variable doesn't have it's own name, a new one is generated and assigned to it. The reason is, that all variables in BIR have to be named.
