Detailed Description

Handler of variables created during decompilation.

Instances of this class have reference object semantics. This class is not meant to be subclassed.

◆ getVariableByName()

ShPtr< Variable > llvmir2hll::VarsHandler::getVariableByName ( const std::string &  varName)

Returns the variable named by varName.

Parameters
[in]varNameName of the requested variable.

If varName is the name of an existing global variable, local variable (including function parameters), or a function, this variable is returned. If there is no variable named varName, this function creates a new one, adds it either to resModule as a global variable (when convertingGlobalVars is true), or into localVars (when convertingGlobalVars is false), and returns it.
