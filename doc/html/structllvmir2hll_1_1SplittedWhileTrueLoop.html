Detailed Description

A representation of a "while true" loop.

Consider a general "while true" loop:

while True:
// Statements (1)
if cond: // Loop end (2)
either nothing or a variable assignment
break or return
// Statements (3)

This class represents a loop splitted into the parts (1) through (3).
